# Studio Frontend Prototype

Pure frontend prototype based on the Figma component direction:

`https://www.figma.com/design/02bqX5Byo9O3VkTSi6NjpV/Untitled?node-id=0-1`

This folder is separate from the Supabase/Next.js app. It contains no backend code.

## Open

Open `index.html` directly in a browser:

```text
/Users/zhouding/Downloads/chihohanstudio/figma-frontend/index.html
```

Or serve the folder:

```bash
python3 -m http.server 4173
```

Then visit:

```text
http://127.0.0.1:4173
```

## Screens

- Lessons page
- Sidebar navigation
- Placeholder Dashboard, Studio Class, Students, and Announcements pages

The Lessons page uses grouped date sections, compact lesson rows, badges, and circular icon buttons.
